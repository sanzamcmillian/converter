# Sort Word

This is the frontend for the Sort Word application, built with React, Cloudscape Components, and React Router. It connects to a FastAPI backend to provide full functionality for sorting a word

## Features

- Input a word and an email
- Client-side validation:
  - Email must be valid
- Sends a POST request to the FastAPI backend
- Displays sorted result dynamically
- Responsive and modern UI with Cloudscape Design System


## Installation
    1. git clone https://github.com/sanzamcmillian/converter.git 
    2. cd frontend/sort-word
    3. npm install
    4. npm run dev
    5. Open your browser and navigate to: http://localhost:5173
